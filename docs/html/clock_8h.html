char PLL_M,
char PLL_N,
CLOCK_PLLR_State PLL_R )

Configures the PLL clock so that it may be used as the System clock.

Parameters
pll_srcselects which clock will drive the PLL
PLL_Mdivsion factor for the main PLL clock. 1 <= PLL_M <= 16.
PLL_Nmain PLL multiplication factor. 8 <= PLLN <= 127.
PLL_Rdivsion factor for the main PLL clock. PLLR = {2, 4, 6, 8}.
Returns
status
Warning
This function can only be called if the PLL clock is disabled. Use CLOCK_isActivated() and CLOCK_desactivateClk() before calling this function.
