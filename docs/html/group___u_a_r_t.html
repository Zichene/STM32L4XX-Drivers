◆ UART_hasData()

uint8_t UART_hasData ( UART_DEVICE_State uart)

Reads and returns the RXNE/RXFNE bit in the UARTx->ISR register.

This determines whether or not data has been received and is ready to be read from the RDR register.

Parameters
uartUART/USART device.
Return values
1Data has been received and is ready to be read.
0Data has not been received.
